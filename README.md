# Linked_List-DataStructure_Application
### 主要方向: 透過所附加的JSON檔用鏈結串列實作以下操作，JSON可使用的欄位為"姓名/學號"，"朋友清單"，"發表過的文章編號"，"文章按讚表"。
1. 讓使用者可以輸入學號"或"姓名。<br>
2. 使用者輸入特定的文章ID可以對該文章進行按讚，如果與該發文者為朋友的關係，則可按讚，否則顯示無權限。<br>

### 簡介: 資料結構鏈結串列實作類資料庫CRUD
